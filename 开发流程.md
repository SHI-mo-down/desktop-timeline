## 项目文件结构

```
DesktopIconSorter/
├── src/
│   ├── main.cpp                 # 入口：控制台/托盘模式
│   ├── core/
│   │   ├── icon_manager.h/cpp   # IFolderView 封装
│   │   ├── grid_system.h/cpp    # DPI感知网格坐标
│   │   ├── sort_engine.h/cpp    # 多分区排序引擎
│   │   └── config_manager.h/cpp # JSON配置管理
│   └── utils/
│       └── com_ptr.h            # COM智能指针
├── ShellExt/
│   ├── shellext.h/cpp           # IExplorerCommand 右键菜单
│   ├── ShellExt.def             # DLL导出
│   └── register.bat             # 注册脚本
├── Hook/                         # 阶段三新增：注入拦截
│   ├── dxgi_proxy.h/cpp         # dxgi.dll 代理
│   ├── iat_hook.h/cpp           # IAT Hook 引擎
│   └── menu_intercept.cpp       # 菜单拦截逻辑
├── Config/                       # 阶段四：C# WPF 配置工具
├── Tests/
│   └── IconSorterTests.cpp      # 验证程序
├── app.manifest                  # DPI感知清单
└── install_menu.bat             # 右键菜单安装
```
